# BALONPI 2026

A football manager simulation game built with React, TypeScript, and Vite.

## Features

- **Squad Management**: View and manage your team with pitch formation view and list view
- **Swipeable Transfer Status**: Swipe left/right on players to change their transfer status (Intransferible, Cedible, En Venta)
- **Match Simulation**: Real-time match simulation with tactical adjustments
- **League System**: Full season simulation with all teams playing their fixtures
- **Player Development**: Players age and develop their skills over seasons
- **Transfer Market**: Buy and sell players with realistic valuations
- **Multi-Season Career**: Play multiple seasons with automatic fixture regeneration
- **Spanish UI**: Fully localized interface in Spanish

## Tech Stack

- React 19
- TypeScript
- Vite
- Tailwind CSS
- IndexedDB (via idb)
- Zustand for state management

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```

## Game Structure

- **Dashboard**: Overview of your club, standings, and next match
- **Squad**: Manage your players, formations, and tactics
- **World**: Browse other teams and the transfer market
- **Fixtures**: View upcoming and past matches
